# 1-st-construction
# Сайт для музыкальной группы

# Что пользователь может делать
Пользователь может купить билеты, посмотреть точное рассписание концертов, а так же изменения в нем (в случае переноса или проблем с площадкой для его проведения) и место проведения при помощи карт на сайте. Так же планируется вкладка с мерчем для покупки.

# Вид сайта
## Главная страница
Фото группы
Рассписание (дата, время, город, адрес, количество оставшихся билетов, кнопка покупки билета)
Небольшое описание группы (может быть небольшая история или песня)
Ссылки на соцсети и контакты для связи

## Покупка билетов
Повторение информации с главной страницы о времени концерта
Карты для простмотра локации концерта
Почта для отправки билета и платеж (?пока муляж?)

## Магазин
Товары с небольшим описанием, картинкой, ценой и количеством оставшегося товара. (При нажатии можно будет посмотреть больше картинок и возможно больше информации, так же оформить заказ)
